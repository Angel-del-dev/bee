We welcome all contributions, including bug fixes, new features, documentation improvements, and ideas.

This project is open source and your help is highly appreciated.

# 🚀 How to Contribute
Fork the repository
Clone your fork
```bash
git clone https://github.com/Angel-del-dev/bee.git bee
```
1. Create a new branch following the branch naming convention
2. Make your changes
3. Commit your changes using the commit message convention
4. Push your branch
5. Open a Pull Request
# 🌿 Branch Naming Convention

All branches must follow this format:
```
type/description
```
## Examples:
+ feature/add-command-execution
+ feature/improve-project-structure
+ chore/implement-todo-code
+ document/modify-readme
+ task/fix-memory-bug
## Rules:
+ Use lowercase
+ Use hyphens (-) to separate words
+ Always include a prefix type followed by /
+ Allowed prefixes:
  * feature/ → for new features
  * chore/ → maintenance tasks, refactors, updates
  * task/ → general work or fixes not tied to features
  * document/ → for documentation
# 💬 Commit Message Convention

All commit messages must include a prefix followed by a colon.
```
type: short description
```
## Examples:
+ feat: add command execution
+ chore: implement todo functionality in memory
+ task: fix memory bug
+ document: update README instructions
## Allowed prefixes:
+ feat: → new features
+ chore: → maintenance tasks, build updates, refactors
+ task: → general fixes or work items
+ document: → documentation changes
## Rules:
+ Use present tense ("add", not "added")
+ Keep messages short and clear
+ No period at the end
# 🛠 Pull Request Guidelines

## Before submitting a PR:
+ Ensure your branch follows the naming convention
+ Ensure commits follow the required format
+ Keep changes focused and small
+ Make sure the project builds/runs correctly
+ Reference related issues when applicable
# 🐛 Reporting Issues

## If you find a bug or want to suggest something:
+ Open an issue
+ Clearly describe the problem or idea
+ Add steps to reproduce if it's a bug
+ Include screenshots or logs if useful
# 🤝 Code of Conduct

Be respectful, constructive, and collaborative.
Harassment or toxic behavior will not be tolerated.
